    artifacts/api-server/src/lib/rate-limit.ts:48.
  - Frontend upload limit says 200 MB, backend enforces 100 MB, and oversized multer errors likely surface as generic 500s instead of 413s. See artifacts/cadstone/src/lib/uploads.ts:3, artifacts/api-server/src/lib/uploads.ts:10,
    artifacts/api-server/src/lib/uploads.ts:237.
  - Lead and daily-log attachment uploads write object storage plus DB rows without the transaction/rollback pattern used elsewhere. Failures can orphan files or partial records. See artifacts/api-server/src/routes/leads.ts:1058,
    artifacts/api-server/src/routes/daily-logs.ts:1775.
  - Agent message handling has no per-user in-flight cap, no dedicated agent rate limit, and request aborts do not cancel downstream Anthropic/tool work. See artifacts/api-server/src/routes/agent.ts:1, artifacts/api-server/src/
    lib/agent/orchestrator.ts:1.
  - MCP HTTP default base URL uses port 5000 if PORT is unset, while the API defaults to 8080. See lib/mcp-server/src/http-transport.ts:132, artifacts/api-server/src/index.ts:11.
